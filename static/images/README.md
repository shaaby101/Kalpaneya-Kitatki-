# Author Images

Place author portrait images in this directory.

## Required Images

- `kuvempu.jpg` - Portrait image of Kuvempu
- `tejaswi.jpg` - Portrait image of Poornachandra Tejaswi

## Image Specifications

- Recommended aspect ratio: 2:3 (portrait orientation)
- Recommended size: 500x750 pixels or similar
- Format: JPG, PNG, or WebP
- The images will be displayed as poster-style covers on the homepage

## Adding New Authors

When adding new authors, update the `image_url` field in the Author table to point to the image filename (e.g., `author_name.jpg`).
